# Documentation

This directory contains project documentation including AI session logs, architecture decisions, and operational procedures.

## Structure

- `ai/` - AI agent-related documentation including session logs and architecture decision records (ADRs)
- `api/` - API documentation and contracts
- `deploy/` - Deployment documentation including access control and infrastructure setup
- `ops/` - Operational documentation including daily reports and incident response procedures
- `security/` - Security documentation including threat models and security procedures

See `AGENTS.md` (repository root) for project operating rules and agent guidelines.
